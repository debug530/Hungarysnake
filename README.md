# 🐍 Hungary Snake Game

一个功能丰富的贪吃蛇网页游戏，专为Hungary项目打造。

## 🎮 在线游玩
- **游戏主页:** https://debug530.github.io/Hungarysnake/
- **测试页面:** https://debug530.github.io/Hungarysnake/test.html

## ✨ 功能特点
- 4种游戏模式：经典、限时、挑战、无尽
- 4种难度级别：简单、中等、困难、专家  
- 4种特殊食物：普通、双倍、加速、减速
- 响应式设计，支持桌面和移动端
- 完整的音效系统和排行榜

## 🛠️ 技术栈
- HTML5 Canvas
- CSS3 + JavaScript
- Web Storage API
- Web Audio API
